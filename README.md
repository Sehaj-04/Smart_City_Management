# Smart City Transport System 🚦

## 📌 Description
This project helps in finding the shortest route and nearest hospitals using graph algorithms.

## 🚀 Features
- Shortest path calculation
- Emergency routing
- Nearby hospital detection
- Traffic-based route suggestions

## 🛠️ Technologies Used
- Python
- Streamlit
- Folium
- Graph Algorithms (Dijkstra)

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
